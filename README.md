# Blockchain-based IDS

A basic intrusion detection system that logs alerts on Ethereum blockchain using Solidity and Python.

## Tech Stack
- Python (Web3.py)
- Solidity
- Ganache
- Remix IDE

## Features
- Rule-based IDS detection
- Smart contract logging
- Immutable blockchain alerts
